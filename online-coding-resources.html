Free Online Coding Resources

I have been asked many times about good online free coding resources. I pretty much have an email that I copy and paste to this question so thought I'd put it online for everyone to look at. I have categorised them into videos, courses, games and tasks/problem solving as I know everyone learns differently. Then below I mention what languages it covers.

These are all resources I have used or know that people have used. I am aware there are tonnes more so don't hate on me if one you love isn't here. It is Open Source so you could do a PR and add it üòú.

VIDEOS

  • egghead.io - This is my go to site for all things JavaScript. Each series has many videos each being a few minutes long, and most have a transcript underneath too.

    Topics Taught: JavaScript, React, Angular, NodeJS. They also do a great series on ES6 and Redux.

  • The Code Player - Video style walkthroughs showing things being built from scratch.

    Topics Taught: JavaScript, HTML5, CSS3, Javascript

COURSES

  • Codecademy - Codecademy is the first place I (and most others) go when learning a new language. The platform revolves around interactive learning; you read a little, type your code right into the browser, and see results immediately.

    Topics Taught: HTML/CSS, JavaScript, Ruby, Python, Java, PHP, Git, Command Line, React, Angular and SQL.

  • Khan Academy - They offer practice exercises, instructional videos, and a personalized learning dashboard that empowers learners to study at their own pace. They also have a few courses for younger kids, too.

    Topics taught: Many

TASKS AND PROBLEM SOLVING

  • tryGit - A really fun and great 15 minute intro to Git.

  • Little Projects on GitHub by Karan Goel - This repo has exmaples of little tasks that commonly come up in tech tests. Including Find PI to the Nth Digit and Fibonacci Sequence. Does also include the solutions. But try not to cheat.

GAMES

  • CodinGame - Practice coding with fun programming challenges.

    Topics taught: JavaScript, C++, Java, Python, Clojure, Perl, Rust and many many more.

  • Flexbox Froggy - Guide a frog and its friends to the lilypad by writing CSS.

    Topics taught: FlexBox.

OTHER

  • CSS Tricks - This blog gives you lots of tips, tricks, and techniques for CSS. As well as Sass, JavaScript, PHP, and more.
